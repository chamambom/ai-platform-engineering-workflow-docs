# Documentation & Knowledge Discovery

Improved access to operational knowledge by using Confluence search via MCP tools to retrieve relevant documentation for unfamiliar services, reducing onboarding time and speeding up troubleshooting.

## Approach

1. **Confluence search via MCP** — Query organisational knowledge base for relevant docs
2. **Service discovery** — Locate documentation for unfamiliar services quickly
3. **Context retrieval** — Pull architecture decisions, runbooks, and operational guides into working context

## Outcomes

- Reduced onboarding time for new services and domains
- Faster troubleshooting through immediate access to relevant documentation
- Less reliance on tribal knowledge and ad-hoc questions
- Operational knowledge surfaced at point of need
